# 在线题库学习考试系统 — 产品介绍

---

## 一、产品简介

本系统是一套面向**建筑检测、职业培训、企业内训**等场景的一体化在线题库学习与考试管理平台。

系统涵盖**在线练习、模拟/正式考试、视频课程、学习资料、证书管理、数据分析**等核心能力，分为用户端（学员）与管理后台（管理员/负责人）两大模块，支持 PC 端和移动端自适应访问，开箱即用。

---

## 二、核心特色

| 特色 | 说明 |
|------|------|
| 智能练习 | 按科目、章节、难度自由组题，答错自动入错题本，支持错题重做与"已掌握"标记 |
| 仿真考试 | 正式考试模式模拟真实考场：倒计时、防切屏监控、异常自动记录、超时强制交卷 |
| 双证体系 | 练习证书（章节通过率达标即可申请）+ 考试证书（正式考试通过自动颁发） |
| 人工+自动批改 | 客观题交卷即出分；含综合简答题的试卷支持管理员在线人工批改，快速评分与关键词匹配评分 |
| 多端适配 | PC、手机、微信浏览器全页面自适应，支持深色/浅色主题切换 |
| 全局搜索 | Ctrl+K 一键搜索题目、资料、公告，快速定位所需内容 |
| 数据分析 | 全平台练习量、正确率、通过率等多维度统计，图表化展示 |

## 三、用户端功能（学员）

### 3.1 工作台
- **问候与日期**：根据当前时间显示"上午好/下午好/晚上好"，展示当前日期
- **学习数据总览**：总做题量、正确率、错题数、考试次数四大核心指标卡片
- **快捷入口**：练习中心、参加考试、错题本、我的证书、视频课程、学习资料、收藏夹、笔记，一键直达
- **公告横幅**：置顶公告以醒目黄色条幅展示，标题+内容一目了然
- **最近考试**：显示最近5次考试记录，包含试卷名称、得分、通过状态
- **学习趋势图**：近30天做题量柱状图，直观展示学习节奏
- **通知中心**：右上角铃铛图标，显示未读消息数量

### 3.2 练习中心
- **科目选择**：按科目卡片选择练习范围，每行显示8个科目卡片（PC端）
- **章节筛选**：选择科目后可进一步选择具体章节
- **题型支持**：单选题、多选题、判断题、综合简答题四种题型
- **答题模式**：
  - 逐题作答模式：每题选择答案后查看正确答案与详细解析
  - 支持上一题/下一题切换
  - 题号导航栏快速跳转
- **即时反馈**：答对显示绿色"正确"，答错显示红色"错误"并展示正确答案
- **解析展示**：每题显示详细解析，帮助理解知识点
- **收藏功能**：做题过程中可一键收藏重点题目
- **笔记功能**：做题过程中可直接添加学习笔记
- **综合题智能评判**：简答题采用相似度智能评判，非简单字符串匹配，容错率更高
- **完成统计**：练习结束后显示本次做题数、正确数、正确率

### 3.3 考试管理
- **模拟考试列表**：
  - 可多次重做，不开启防作弊
  - 显示考试时长、总分、及格分、题目数量
  - 适合日常练习和考前冲刺
- **正式考试列表**：
  - 仅一次机会（不及格可申请重考，需管理员审批）
  - 显示已参加状态和成绩
  - 支持邀请码试卷（输入正确邀请码方可进入）
- **考试界面（仿真模式）**：
  - 考前准备页：显示考试须知、时长、总分
  - 倒计时计时器，醒目显示剩余时间
  - 题号导航面板：可快速跳转至任意题目，已答/未答/标记状态一目了然
  - 标记功能：可标记不确定的题目，方便后续回查
  - 交卷确认：提前交卷需二次确认
  - 超时自动交卷
- **防作弊监控（正式考试）**：
  - 切屏检测：切换浏览器标签页自动记录
  - 页面离开检测：尝试离开考试页面弹窗警告
  - URL跳转检测：直接修改地址栏URL会被记录
  - 异常次数超限自动交卷
  - 异常操作实时记录至异常报告
- **考试结果**：
  - 交卷后显示得分、通过状态、用时
  - 客观题即时出分
  - 含综合题的试卷显示"待批改"状态

### 3.4 我的成绩
- **成绩列表**：查看全部考试记录，包含试卷名称、得分、用时、通过状态
- **搜索筛选**：支持按科目/关键词搜索
- **成绩详情**：点击记录查看每道题的作答情况、正确答案、解析
- **统计卡片**：总考试次数、通过次数、平均分、最高分

### 3.5 练习记录
- **记录列表**：记录每次练习的科目、题数、正确率、用时、日期
- **详情查看**：点击记录可查看详细答题历史，包含每题的作答情况
- **删除功能**：可删除不需要的练习记录

### 3.6 错题本
- **自动收集**：练习与考试中的错题自动同步至错题本
- **科目分类**：按科目分类展示错题
- **复习模式**：点击"开始复习"进入逐题重做模式
  - 逐题重做错题，答对自动标记为"已掌握"
  - 答错继续留在错题本
  - 显示当前掌握进度
- **错题详情**：可查看题目内容、正确答案、解析、错误次数
- **笔记功能**：可在错题上添加复习笔记
- **移除功能**：已掌握的题目可手动移除

### 3.7 收藏夹
- **收藏列表**：收纳用户标记的重点题目
- **卡片式展示**：每个收藏题目以卡片形式展示，包含题目内容、科目、题型
- **操作功能**：
  - 点击可查看完整题目内容和答案解析
  - 可取消收藏
  - 可直接进入做题模式
- **科目筛选**：按科目分类查看

### 3.8 笔记管理
- **随题笔记**：做题时直接关联题目记录笔记
- **独立笔记**：自由新建笔记，支持标题、正文内容
- **搜索功能**：支持关键词搜索笔记内容
- **编辑删除**：支持修改和删除已有笔记

### 3.9 学习资料
- **资料浏览**：按科目分类浏览管理员上传的文档资料
- **文件类型**：支持 PDF、Word、Excel、PPT、图片等常见格式
- **在线预览**：PDF 文件支持在线预览，无需下载
- **搜索功能**：支持按关键词搜索资料
- **下载功能**：支持下载资料文件

### 3.10 视频课程
- **课程浏览**：按科目分类浏览视频课程
- **课程卡片**：每个视频显示封面缩略图、标题、时长
- **播放功能**：
  - 点击即可播放视频
  - 支持全屏播放
  - 自动记录播放进度，支持断点续播
- **搜索功能**：支持按关键词搜索课程

### 3.11 证书中心
- **练习证书**：
  - 章节练习正确率达到 60% 即可在线申请
  - 系统自动校验章节通过率
  - 申请后即时颁发
- **考试证书**：
  - 正式考试通过后自动颁发
  - 未自动颁发时可手动申请，系统校验考试成绩
- **证书展示**：
  - 已颁发证书以卡片形式展示
  - 显示证书名称、颁发时间、证书编号
  - 证书被撤销时页面显示状态提醒
- **申请记录**：查看历史申请记录和状态

### 3.12 全局搜索
- **快捷键触发**：Ctrl+K 快速打开搜索面板
- **多类型搜索**：同时搜索题目、资料、公告
- **结果分类**：搜索结果按类型分组展示（题目/资料/公告）
- **快速跳转**：点击结果直接跳转到对应页面

### 3.13 通知中心
- **消息类型**：考试提醒、批改完成、证书变动、新公告等
- **未读标记**：未读消息以红点/数字提示
- **消息列表**：按时间倒序展示，显示标题、内容、时间
- **全部已读**：一键标记所有消息为已读

### 3.14 个人中心
- **基本信息**：查看和修改用户名、姓名等个人信息
- **头像显示**：自动根据用户名首字母生成彩色头像

### 3.15 深色模式
- **一键切换**：顶栏月亮/太阳图标切换深色/浅色模式
- **全局适配**：所有页面、组件、弹窗均适配深色主题
- **记忆功能**：切换后自动保存偏好，下次登录保持

## 四、管理后台功能（管理员/负责人）

### 4.1 管理工作台
- **平台数据总览**：总用户数、题库总量、待处理反馈、异常报告四大指标
- **快捷入口**：题库管理、试卷管理、用户管理、数据统计、视频课程、资料管理、证书管理、公告管理
- **系统概览**：实时显示关键运营数据

### 4.2 科目章节管理
- **科目管理**：
  - 新增、编辑、删除学科分类
  - 为每个科目设置描述信息
- **章节管理**：
  - 为每个科目搭建章节结构
  - 章节作为题库、试卷、资料的统一分类载体
  - 支持多级章节树形结构

### 4.3 题库管理
- **多维筛选**：按科目/章节/题型/难度多维筛选
- **题目列表**：卡片式展示题目内容、题型、科目、难度
- **手动建题**：
  - 录入题干（支持富文本）
  - 设置选项（单选/多选/判断）
  - 设置正确答案和详细解析
  - 支持上传题目配图
- **批量导入**：
  - 提供标准 Excel 模板下载
  - 批量上传 Excel 文件录入题目
  - 导入失败时展示错误行号与原因，便于修正
- **编辑删除**：支持修改和删除已有题目
- **题型支持**：单选题、多选题、判断题、综合简答题

### 4.4 试卷管理
- **试卷列表**：查看所有已创建试卷，区分模拟/正式类型
- **创建试卷**：
  - 设置试卷名称、所属科目
  - 选择考试类型：模拟考试 / 正式考试
  - 设置考试时长（分钟）
  - 设置总分和及格分数
- **组卷方式**：
  - 自动组卷：按科目/题型/难度/数量自动选题
  - 手动组卷：逐题挑选指定题目
- **邀请码模式**：开启后需输入正确邀请码方可参加
- **状态控制**：启用/禁用试卷
- **成绩查看**：查看已参加该试卷的学员成绩列表

### 4.5 人工批改
- **待批改列表**：统一收纳含综合简答题的待批改试卷
- **在线阅卷**：
  - 逐题查看学员答案
  - 对照参考答案手动打分
  - 支持设置每题分值
- **快速评分**：
  - 全部满分：一键将所有综合题设为满分
  - 关键词匹配评分：学员答案含参考答案关键词即得满分
- **批改完成**：成绩自动同步至学员端

### 4.6 重考审批
- **申请列表**：查看学员提交的重考申请
- **审批操作**：通过/拒绝重考申请
- **申请理由**：查看学员提交的重考原因
- **审批记录**：记录审批时间和结果

### 4.7 异常报告
- **异常汇总**：汇总正式考试中的所有异常操作记录
- **异常类型**：切屏检测、页面离开、URL跳转等
- **详情查看**：点击查看具体异常时间、类型、次数
- **考试有效性判定**：管理员可判定异常考试是否有效

### 4.8 题目反馈
- **反馈列表**：接收学员提交的题目疑问/错误反馈
- **反馈详情**：查看学员反馈的具体内容
- **处理功能**：标记反馈为已处理，修正题目内容
- **进度跟踪**：跟踪反馈处理进度

### 4.9 资料管理
- **资料上传**：上传学习资料文件
- **文件类型**：支持 PDF、Word、Excel、PPT、图片、压缩包等格式
- **分类管理**：按科目统一分类管理
- **编辑删除**：支持修改资料信息和删除资料
- **直接选择文件**：简洁的文件选择界面，点击即可上传

### 4.10 视频课程管理
- **视频上传**：
  - 直接选择本地视频文件上传
  - 支持 MP4、AVI、MOV 等常见格式
  - 自动提取视频标题（默认为文件名）
- **视频信息**：设置标题、描述、所属科目
- **编辑删除**：支持修改视频信息和删除视频
- **列表展示**：卡片式展示视频封面、标题、时长

### 4.11 公告管理
- **公告列表**：卡片式展示所有公告
- **创建公告**：设置标题、内容、是否置顶、是否立即发布
- **编辑公告**：修改已有公告内容
- **发布控制**：支持发布/撤回公告
- **置顶功能**：置顶公告登录时弹窗提醒
- **统计概览**：已发布数、草稿数、置顶数

### 4.12 证书管理
- **证书模板**：管理练习证书模板、考试证书模板
- **已颁发证书**：查看全平台已颁发证书列表，显示证书名称、用户名、编号、日期
- **撤销功能**：支持撤销已颁发证书（状态实时同步至用户端）
- **证书申请审批**：审批学员的证书申请

### 4.13 用户管理
- **用户列表**：查看全体注册用户列表
- **搜索筛选**：按用户名/姓名搜索
- **状态管理**：启用/禁用用户账号
- **用户详情**：查看单个用户的学习、考试明细数据
- **admin 保护**：admin 账号不可被禁用

### 4.14 角色管理
- **角色列表**：查看所有角色
- **自定义角色**：支持创建自定义角色（学生、实验员、负责人等）
- **角色编辑**：修改角色名称和描述

### 4.15 数据分析
- **多维度统计**：
  - 全平台用户练习量统计
  - 答题正确率分析
  - 考试通过率趋势
  - 科目维度数据对比
- **图表展示**：柱状图、折线图等可视化图表
- **学习热力图**：按日/周/月展示学习活跃度

### 4.16 操作日志
- **自动留痕**：所有后台操作自动记录
- **日志内容**：记录操作人、操作时间、操作类型、操作详情
- **查询筛选**：按时间/操作人/操作类型筛选
- **审计支持**：支持问题溯源与安全审计

## 五、安全特性

| 安全维度 | 说明 |
|---------|------|
| 账号安全 | 密码加密存储（bcrypt），多次登录失败自动锁定 |
| 考试防作弊 | 正式考试切屏检测、页面离开检测、URL跳转检测、异常次数超限自动交卷 |
| 路由守卫 | 正式考试中尝试离开页面弹窗确认，离开自动记录异常 |
| 操作审计 | 后台所有操作自动留痕，支持操作日志查询 |
| 接口安全 | JWT Token 认证，RBAC 权限控制 |
| 数据防护 | 接口防刷、XSS/SQL 注入防护、敏感数据加密 |
| 权限隔离 | 管理员/负责人/学员三级权限，不同角色看到不同菜单和功能 |

## 六、技术架构

| 项目 | 技术选型 |
|------|---------|
| 前端 | Vue 3 + Element Plus + Vite |
| 后端 | FastAPI (Python) + SQLAlchemy |
| 数据库 | SQLite（可迁移至 PostgreSQL / MySQL） |
| 认证 | JWT Token + RBAC 权限控制 |
| 文件存储 | 本地存储（可迁移至 OSS / S3） |
| 样式 | CSS 变量 + 深色/浅色主题系统 |
| 图表 | ECharts 数据可视化 |
| 响应式 | CSS Media Query 多端自适应 |

### 部署方式
- **单机部署**：前后端同机运行，适合中小规模
- **分布式部署**：前后端分离，数据库独立，支持负载均衡
- **容器化**：支持 Docker + Docker Compose 一键部署
- **外网访问**：支持 Cloudflare Tunnel / SSH Tunnel / 公网IP直连等多种外网访问方式

## 七、多端适配

- **PC 端**：Chrome、Edge、Firefox、Safari 主流浏览器，完整功能体验
- **移动端**：手机浏览器、微信内置浏览器，全页面自适应
  - 底部导航栏：首页/练习/考试/错题/更多
  - 卡片式布局：自动调整列数
  - 弹窗自适应：92vw 宽度适配小屏
- **深色模式**：支持一键切换，暗色场景下所有元素清晰可见，包括侧边栏、卡片、图表
